# Лабораторная работа 8: Асинхронный сервис расчёта

**Тема:** Оценка стоимости облачного хостинга для веб-проекта  
**Студент:** Пронин Вячеслав Константинович, ИУ5Ц-73Б  
**Ветка:** `async_web_service`

#### Цель работы
Создание асинхронного сервиса для отложенного расчёта стоимости облачного хостинга.

#### Задание
Разработка второго простого асинхронного сервиса на Go с одним http-методом для выполнения отложенного действия в системе оценки стоимости облачного хостинга.

**Требования:**
1. Сервис написан на Go (основной бэкенд - Django)
2. Один HTTP-метод `POST /calculate`
3. Принимает JSON с полем `calculation_id`
4. Выполняет расчёт с задержкой 5-10 секунд
5. Возвращает результат расчёта (случайный или вычисляемый)
6. Взаимодействует с основным веб-сервисом через HTTP

#### Технологии
- Go 1.21+
- Gorilla Mux (роутинг)
- Docker
- HTTP клиент
